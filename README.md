### # 猫猫Bot-NTQQ!


#### 介绍
这是一个基于QQ开放平台（q.qq.com）提供的 API 搭建的大语言模型机器人，由  **北京一叶知秋科技工作室（有限合伙 ）** 开发，支持 DeepSeek、通义千问等大模型。
